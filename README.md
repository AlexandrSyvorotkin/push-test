# Push Test — PWA

Тестовое приложение для проверки PWA и push-уведомлений на телефоне.

## Локальный запуск

```bash
npm run dev
```

## Сборка и деплой

```bash
npm run build
```

После сборки папка `dist` содержит готовое приложение.

### Варианты хостинга

1. **Vercel** — бесплатно, подходит для React/Vite:
   - Зарегистрируйтесь на [vercel.com](https://vercel.com)
   - Подключите репозиторий или загрузите `dist` через CLI
   - Команда: `npx vercel deploy --prebuilt`

2. **Netlify** — бесплатно:
   - Зарегистрируйтесь на [netlify.com](https://netlify.com)
   - Загрузите папку `dist` или подключите Git

3. **GitHub Pages** — бесплатно:
   - Залейте проект на GitHub
   - В настройках репозитория: Pages → Source: GitHub Actions
   - Или используйте `gh-pages` для деплоя из `dist`

4. **Любой статический хостинг** — загрузите содержимое `dist`.

## Установка как PWA на телефоне

1. Откройте сайт в браузере (Chrome, Safari, Edge).
2. **Android (Chrome):** «Добавить на главный экран» или «Установить приложение».
3. **Android (Samsung):** Меню → «Добавить страницу на главный экран».
4. **iPhone (Safari):** Кнопка «Поделиться» → «На экран „Домой“».

## Важно

- PWA работает только по **HTTPS** (или localhost).
- Для уведомлений сначала нужно разрешить их в браузере.
- Уведомления могут не показываться, если приложение не было открыто хотя бы раз.
